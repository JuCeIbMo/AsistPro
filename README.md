## Key Features

- **Instant load** ‚Äì Optimized images & Next.js dynamic imports
- **Fully responsive** ‚Äì Mobile-first layout that scales to 4K
- **Framer Motion animations** ‚Äì Hero reveal, logo ticker, parallax mockups, fade-in testimonials, and more
- **Reusable components** ‚Äì Header, Button, Section, TestimonialCard, etc.
- **Static & dynamic images** ‚Äì Next `<Image>` component with blur placeholders

## Setup & Installation

### 1. Clone the repository

```bash
git clone https://github.com/pedroestevaodev/stunning-ia-lp.git
cd stunning-ia-lp
```

### 2. Install dependencies

```bash
$ npm install
```

### 3. Start the development server

```bash
$ npm run dev
```

Open your browser and visit `http://localhost:3000` to see the project in action.

You can start editing the homepage by modifying the `app/page.tsx` file. The browser will automatically update as you make changes to the code.
